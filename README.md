# Daily Reflection Tree Assignment

This repository contains Part A (Deterministic Reflection Tree) and Part B (CLI Agent).

## Structure
- tree/reflection-tree.tsv
- tree/tree-diagram.md
- agent/reflection_agent.py
- transcripts/
- write-up.md

## Run Agent

```bash
cd agent
python reflection_agent.py
```

## AI Usage
Used ChatGPT for:
- psychology research references
- refining fixed options for questions
- testing deterministic branching
- improving reflection tone without moralizing

No LLM is used at runtime.
